aria-autocomplete

External resource for testing aria-autocomplete

Tested on:

  • iOS 9.3 and VoiceOver
    • Mostly works: Voiceover reads "button", which is not a clear indication that this is an autocomplete field. When the user activates the button, VoiceOver reads "combo box". The keyboard appears on the screen, and as the user types, values change in the combo box and the default value is read automatically during typing. There are a few cases when several items begin with the same letter, the next item is not read as the user types.
