Simple ARIA Toolbar Widget

A generic pattern for "toolbars" which offer buttons for manipulating content in some way. In this case, it's sorting a list.

Sorting crab names

  • Fiddler crab
  • Hermit crab
  • Red crab
  • Robber crab
  • Sponge crab
  • Yeti crab

Source: heydonworks.com

Tested on:

  • iOS 9.31, VoiceOver
    • Results: VoiceOver does not announce the presence of the toolbar, but content inside of the toolbar reads properly.
  • Windows 7, JAWS 16
    • Results: JAWS announces "Sorting options toolbar with 2 buttons".
